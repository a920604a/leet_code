一般形式就是求極值，核心思想是窮舉 + 重疊子問題。狀態、選擇、base case

## 動態規劃框架
```cpp
# 初始化 base case
dp[0][0][...] = base
# 进行状态转移
for 状态1 in 状态1的所有取值：
    for 状态2 in 状态2的所有取值：
        for ...
            dp[状态1][状态2][...] = 求最值(选择1，选择2...)
```

## 壓縮動態規劃
注意到dp[i][j] 都是由 dp[i-1][..] 轉移過來的，之前的數據都不需要了，所以可以將二維壓縮成一維，節省空間複雜度。
