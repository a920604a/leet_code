## [problem](https://leetcode.com/problems/bulb-switcher/)

## solution 
```c++
class Solution {
public:
    int bulbSwitch(int n) {
        return sqrt(n);
    }
};
```
## analysis
- time complexity `O(1)`
- space complexity `O(1)`
